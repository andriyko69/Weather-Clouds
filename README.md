# Weather Clouds

Weather Clouds adds two simple cloud blocks that create localized weather.

## Features

- Rain Cloud – produces rain below the block
- Snow Cloud – produces snow below the block
- Uses vanilla weather rendering (same visuals as normal rain/snow)
- Snow Clouds can place and accumulate snow layers below them

The goal of the mod is simple: place clouds and create small weather zones wherever you want.

## Credits

Cloud textures are based on assets from [The Aether](https://modrinth.com/mod/aether).

The idea and general concept of localized cloud weather was inspired by [The Twilight Forest](https://www.curseforge.com/minecraft/mc-mods/the-twilight-forest) mod.
